MASTER OF SCIENCE IN ARTIFICIAL INTELLIGENCE

CORE COURSES (15 Credits)

Introduction to Artificial Intelligence

MSAI-501
3 Credits

Comprehensive introduction to AI concepts, history, and applications. Students explore search algorithms, knowledge representation, AI planning, and intelligent agent design through hands-on projects and case studies.

Learning Objectives

  • Understand fundamental AI concepts and terminology
  • Implement basic search algorithms (BFS, DFS, A*)
  • Design knowledge representation systems
  • Create simple intelligent agents
  • Analyze AI applications in various domains

Key Topics

Search Algorithms
Knowledge Representation
Expert Systems
Intelligent Agents
AI Planning
Problem Solving

Assessment

Programming Projects 40%
Midterm Exam 25%
Final Exam 25%
Case Studies 10%

Prerequisites

None - Designed for students from any academic background

Resources

  • Russell & Norvig: "Artificial Intelligence: A Modern Approach"
  • Python programming environment
  • AI simulation tools and frameworks

Machine Learning & Data Mining

MSAI-502
3 Credits

  • Apply ML to domain-specific problems
